# Curriculum Sources

Giáo trình trong web được biên soạn lại bằng tiếng Việt, dựa trên các nguồn chính thức/uy tín sau. Nội dung không sao chép nguyên văn; app diễn giải thành bài học, bài tập và milestone project.

## Python Foundation

- Python Tutorial: https://docs.python.org/3/tutorial/
- Các phần dùng làm nền: data structures, functions, errors/exceptions, modules, virtual environments.

## Pandas

- 10 minutes to pandas: https://pandas.pydata.org/docs/user_guide/10min.html
- pandas User Guide: IO tools, missing data, indexing/selecting data, groupby.

## EDA & Visualization

- pandas visualization guide: https://pandas.pydata.org/docs/user_guide/visualization.html
- Matplotlib tutorials: https://matplotlib.org/stable/tutorials/index.html
- Seaborn tutorial: https://seaborn.pydata.org/tutorial.html
- UCI Student Performance Dataset: https://archive.ics.uci.edu/dataset/320/student+performance

## Scikit-learn

- scikit-learn User Guide: https://scikit-learn.org/stable/user_guide.html
- Các phần dùng làm nền: preprocessing, model selection, metrics, compose/ColumnTransformer, Pipeline.

## FastAPI

- FastAPI Tutorial - User Guide: https://fastapi.tiangolo.com/tutorial/
- Các phần dùng làm nền: request body, Pydantic model, response model, SQL databases, testing.

## Streamlit

- Streamlit tutorials: https://docs.streamlit.io/develop/tutorials
- Các phần dùng làm nền: widgets, elements, execution flow, multipage apps, deployment concepts.

## Deploy

- Vercel Vite docs: https://vercel.com/docs/frameworks/frontend/vite
- Vercel Postgres docs: https://vercel.com/docs/postgres

## Ghi chú học thuật

- Dataset mẫu trong `public/data/students.csv` chỉ dùng để học pipeline và demo. Khi làm đồ án nghiêm túc, nên dùng UCI Student Performance Dataset hoặc dataset thật có giấy phép/đồng ý phù hợp.
- Prediction trong project chỉ dùng cho mục đích học tập, không dùng để ra quyết định thật về sinh viên.
